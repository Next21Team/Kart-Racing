# Kart Racing

_[English](README.md) | **Русский**_

[![Kart Racing](https://img.youtube.com/vi/THKErtYjpUo/0.jpg)](https://youtu.be/THKErtYjpUo)

AMX Mod X плагин для Counter-Strike.

Kart Racing — аркадные гонки на безумных скоростях. На гоночных трассах разбросаны предметы, позволяющие временно улучшать свой карт и строить ловушки для соперников. Также на выбор доступно множество персонажей с возможностью перекраски.

## Управление
Кроме стандартных кнопок передвижения машины по трассе, в игре доступны следующие действия:

* **Использовать собранный предмет** - "use" (E).
* **Скольжение** - Прыжок (пробел) + вперед + поворот.
* **Прыжок** - Зажать "duck" (Ctrl) и через время отпустить (сила прыжка зависит от времени зажатия клавиши).
* **Посмотреть назад** - Зажать правую кнопку мыши или нажать на "drop" (G).
* **Сбросить машину к чекпоинту** - Зажать и удерживать "reload" (R).
* **Звуковой сигнал** - "impulse 100" (F).
* **Перейти в меню выбора персонажа** - "chooseteam" (M).

## Квары
- ```kart_laps_num "3"``` Количество кругов в заезде.
- ```kart_reverse_available "0"``` Включить заезд в обратную сторону.
- ```kart_start_waiting_time "45"``` Время (секунды) ожидания других игроков перед началом игры.
- ```kart_ending_time "25"``` Время (секунды) ожидания других игроков после финиширования.
